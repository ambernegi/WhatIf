title: Data Management API

---

The Data Management API (WIP DM v3) is an Autodesk Platform Services (APS) API for cloud storage and file management in Autodesk construction and design workflows.

**WIP** refers to **Work In Progress** data management.

**Base URL:** `https://developer.api.autodesk.com/wipdata-serv`
