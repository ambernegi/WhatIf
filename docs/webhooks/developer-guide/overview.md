# Overview

Webhooks let you receive real-time HTTP callbacks when events occur in your Autodesk Platform Services account. Build reactive, event-driven integrations without polling.

## When to use this API

- Triggering downstream workflows when a model translation completes
- Notifying collaborators when project files are updated
- Building event-driven pipelines that react to APS service state changes

## Capabilities

- Subscribe to events across APS services
- Filter by event type, resource, and scope
- Retry delivery with configurable backoff on failures
