# Overview

The Metrics & Analytics API gives you visibility into how your APS integrations are performing — API call volumes, error rates, and latency breakdowns. Use it to monitor, debug, and optimize your applications.

## When to use this API

- Monitoring API call volumes and error rates across your integrations
- Debugging performance issues with latency and p99 metrics
- Exporting usage data for custom dashboards and alerting pipelines

## Capabilities

- Query usage metrics by time range and resource
- Monitor error rates and p99 latency per endpoint
- Export data for dashboards and alerting

## Beta

This API is currently in **beta**. Endpoints and response shapes may change before general availability.
