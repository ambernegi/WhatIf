# Overview

The Viewer API lets you embed and interact with 3D models directly in your web applications. It supports CAD, BIM, and point cloud formats with no browser plugins required.

## When to use this API

- Rendering 3D models in the browser for design review workflows
- Building custom viewing experiences with selection, measurement, and annotation tools
- Extending the viewer with your own overlays, custom tools, and UI panels

## Capabilities

- Hardware-accelerated rendering with no plugins
- Interact with model data: select, isolate, and measure geometry
- Access model properties and metadata programmatically
- Extend with custom extensions and tool overlays
